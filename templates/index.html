One-time Google API setup

You only need to do this once. After that, the token is stored locally.

  1. 1 Go to console.cloud.google.com → create a project → search Blogger API v3 and enable it.
  2. 2 Under Credentials, create OAuth 2.0 Client ID (type: Web application). Add http://localhost:5000/blogger/callback as an authorized redirect URI.
  3. 3 Download the JSON file and save it as config/credentials.json inside the blogger_tool folder.
  4. 4 Click Connect with Google below. You'll be redirected to Google to authorize, then returned here.

Personal Blog Management

My Blogging Helper

A private web app for writing, formatting, and managing posts on Blogger.com — with AI smart formatting, bulk editing tools, and direct API access to your blog.

Created April 18, 2026 Built with VS Code & Claude Code Python · Flask · Google APIs
How It's Organised
📝
Blog Editor

Connect to Blogger via OAuth. Browse, edit, and push posts back live. Filter by year, one-click AI reformat.

🛠️
Utilities

Single Post tools (Blog Formatter, Image Cleaner) and Bulk Operations (Link Cleanup, Bulk Reformat) in one tab.

ℹ️
About

This page — feature overview, input syntax reference, and build info.

Features
✍️
Blog Formatter

Paste plain text or markdown → get fully-styled inline-CSS Blogger HTML. 7 color themes. Smart Format uses Gemini AI to auto-structure sections, KPI stats, lists and tables.

Utilities → Single Post
🖼️
Image Cleaner

Paste any Blogger HTML → extracts all <a><img> blocks, strips float:left and margin styles, previews thumbnails, and copies clean HTML.

Utilities → Single Post
📝
Blog Editor

OAuth connection to Blogger API v3. Browse posts by year, load HTML, one-click ⚡ Reformat + Images, live preview, and save back. Labels/tags preserved on save.

Blog Editor tab
🔗
Link Cleanup

Scans every post and removes stray tags (editable — defaults to the old Google Fonts <link>). Processes one post per 1.5 s to stay under API limits. Fix Next 25 or Fix All with cancel.

Utilities → Bulk Operations
✨
Bulk Reformat + Images

Select posts with checkboxes → AI reformats + extracts images → pushes back 10 at a time. Icons show ✅ already done / 🔄 different theme / ○ not yet formatted. Cancel at any time.

Utilities → Bulk Operations
Gemini AI

Pick from Gemini 2.5 Flash, 2.5 Flash Lite, 3 Flash (preview), or 3.1 Flash Lite via the ⚙ panel. Each model has its own daily quota — switch if you hit a limit. Key saved locally.

⚙ settings panel — available on Blog Editor and Utilities
