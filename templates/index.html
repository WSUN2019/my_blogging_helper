One-time Google API setup

You only need to do this once. After that, the token is stored locally.

  1. 1 Go to console.cloud.google.com → create a project → search Blogger API v3 and enable it.
  2. 2 Under Credentials, create OAuth 2.0 Client ID (type: Web application). Add http://localhost:5000/blogger/callback as an authorized redirect URI.
  3. 3 Download the JSON file and save it as config/credentials.json inside the blogger_tool folder.
  4. 4 Click Connect with Google below. You'll be redirected to Google to authorize, then returned here.

About

A private local tool for writing, formatting, and managing posts on Blogger.com. Runs on your machine — no data leaves except API calls to Google.

Blog Editor
Browse & Edit Connect via Google OAuth. Browse posts by year, search by title, load HTML, edit, and push changes back to Blogger.
Reformat + Images One-click: AI smart-formats the post with the active theme, extracts images, and appends them at the end.
Format with Theme Pick a color theme from a popup and re-render the post HTML in place — no page change needed.
Post Versions Save snapshots of any post (💾 Save Version). Toggle between the live version and saved snapshots. A badge shows live / modified / version state.
Utilities — Single Post
Blog Formatter Paste plain text or markdown, choose a theme, and get fully styled Blogger-ready HTML with inlined CSS.
Smart Format (AI) Sends your text to Google Gemini to auto-structure sections, KPI stats, lists, and tables before rendering.
Image Cleaner Paste Blogger HTML — extracts all <a><img> blocks and strips float/margin styles.
Utilities — Bulk Operations
Link Cleanup Scan all posts and remove a stray HTML tag (e.g. a Google Fonts <link>) in bulk.
Bulk Reformat AI-reformat + extract images across multiple posts, 10 at a time, and push results back to Blogger.
Utilities — Backup Posts
Generate Backup Pulls all posts from Blogger API and saves them as a dated .atom file (e.g. myblog_20260421.atom) in blog_backup/. Nothing leaves your machine.
Browse Backups Switch between saved backup files. Filter by year or label, preview post HTML, copy to clipboard.
